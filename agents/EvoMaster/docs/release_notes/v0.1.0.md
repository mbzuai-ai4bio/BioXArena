# EvoMaster v0.1.0 Release Notes

> Release Date: 2026-03-23 | Previous Version: v0.0.2

EvoMaster v0.1.0 adds many new Playground examples, including the verified SciMaster family: ML-Master, ML-Master 2.0, X-Master, and Browse-Master. It also introduces a Feishu (Lark) smart assistant: users can start, monitor experiments, and create agents via Feishu and natural language. After configuring Feishu permissions and keys, run `python -m evomaster.interface.feishu` to use it. TypeScript Skill support has been added; see `playground/minimal_openclaw_skill` for details. Several bugs have been fixed and some code structures have been improved.

---

## New Playgrounds

- **ML-Master** (`playground/ml_master/`) — Autonomous machine learning agent
- **ML-Master 2.0** (`playground/ml_master_2/`) — Autonomous machine learning agent
- **X-Master** (`playground/x_master/`) — General-purpose research agent
- **Browse-Master** (`playground/browse_master/`) — Web search agent
- **Agent Builder** (`playground/agent_builder/`) — Meta-agent for automatic agent construction, usable by Feishu dialog agents
- **Chat Agent / MagiClaw** (`playground/chat_agent/`) — Feishu dialog agent prototype; the full MagiClaw will be released in v0.1.0
- **Minimal Bohrium** (`playground/minimal_bohrium/`) — Bohrium scientific computing platform integration example
- **Minimal OpenClaw Skill** (`playground/minimal_openclaw_skill/`) — TypeScript skill translation and integration example
