# EvoMaster v0.1.0 发布说明

> 发布日期：2026-03-23 | 上一版本：v0.0.2

EvoMaster v0.1.0 新增了大量 Playground 示例，其中包括已验证的 SciMaster 系列智能体 ML-Master、ML-Master 2.0、X-Master 和 Browse-Master。同时，我们增加了一个飞书智能助手，用户可以通过飞书和自然语言来启动、监控实验，创建智能体等，只需配置飞书权限和密钥后简单使用 `python -m evomaster.interface.feishu` 即可。新增了对TypeScript语言的Skill的支持，详见`playground/minimal_openclaw_skill`。此外，我们还修复了一些已知的 Bug。优化了部分代码结构。

---

## 新增 Playground

- **ML-Master** (`playground/ml_master/`) — 自主机器学习智能体
- **ML-Master 2.0** (`playground/ml_master_2/`) — 自主机器学习智能体
- **X-Master** (`playground/x_master/`) — 通用科研智能体
- **Browse-Master** (`playground/browse_master/`) — 网页搜索智能体
- **Agent Builder** (`playground/agent_builder/`) — 元智能体，可被飞书对话智能体用于自动构建其他 Agent
- **Chat Agent / MagiClaw** (`playground/chat_agent/`) — 飞书对话智能体原型，完全体MagiClaw将会在`v0.1.0`版本发布和开源
- **Minimal Bohrium** (`playground/minimal_bohrium/`) — Bohrium 科学计算平台集成示例
- **Minimal OpenClaw Skill** (`playground/minimal_openclaw_skill/`) — TypeScript 语言技能转译和接入示例

---

## 飞书智能助手

- 新增飞书 Bot 集成，用户可通过飞书与 EvoMaster 进行自然语言交互
- 飞书智能体助手完全体MagiClaw将在`v0.2.0`版本发布和开源

## TypeScript Skill支持

- 新增了对TypeScript语言的Skill的快速接入支持，代码和使用指南详见`playground/minimal_openclaw_skill`

---

## Bug 修复与代码改进

- 修复了所有已知的bug
- 优化了部分代码的底层实现，但不影响表层代码接口
